# Iris Flower Classification using Machine Learning

## Project Overview

This project classifies Iris flowers into three species:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

A Decision Tree Classifier is used to train the model and predict flower species based on flower measurements.


## Dataset

Dataset Name:

Iris.csv

Features:

- SepalLengthCm
- SepalWidthCm
- PetalLengthCm
- PetalWidthCm

Target:

- Species

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

## Steps Performed

- Loaded dataset
- Checked missing values
- Split dataset into training and testing data
- Trained Decision Tree Classifier
- Predicted flower species
- Calculated model accuracy
- Generated graphs
